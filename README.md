# Red Horizon

## Управление

| Клавиша | Действие |
|---------|----------|
| W / ↑ | Движение вперёд |
| S / ↓ | Движение назад |
| A / ← | Движение влево |
| D / → | Движение вправо |
| Shift | Ускорение (x1.5) |
| F | Вкл/выкл фонарик |
| Esc | Выход |

## Создание карты

Карта задаётся текстовым файлом `map.txt` в корне проекта.

### Символы

| Символ | Описание |
|--------|----------|
| `&` | Стена (кирпич) |
| `@` | Стена (зелёная) |
| `.` | Пол |
| `0` | Пол (доски) |
| `P` | Стартовая позиция игрока |
| ` ` | Пустота (игнорируется) |

## Сборка

```bash
make
```

Требования: [raylib](https://www.raylib.com/), g++, Linux (X11)
