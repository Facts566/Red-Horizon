## About

**Red Horizon** — a first-person horror game built on the **Vulkan** rendering API. You find yourself trapped in a surreal, decaying reality where the boundaries between nightmare and waking life have dissolved. Explore abandoned corridors, solve cryptic puzzles, and survive encounters with entities that defy comprehension.

> *"The horizon was always red. It was never the sunset."*

---

## Features

- **Vulkan Rendering** — modern, low-level GPU API for high-performance graphics
- **Dynamic Lighting System** — real-time shadows and volumetric effects that shift with your sanity
- **Procedural Atmosphere** — environments that subtly change, ensuring no two playthroughs feel identical
- **Immersive Audio Design** — spatial sound that makes you question what's behind you
- **Physics-Based Interaction** — every object can be examined, thrown, or used

## Getting Started

### Prerequisites

- **OS:** Windows 10/11 (64-bit)
- **GPU:** Vulkan 1.2 compatible graphics card
- **Compiler:** MSVC 2019+ or Clang 12+
- **CMake:** 3.20+

### Build

```bash
# Clone the repository
git clone https://github.com/Facts566/Red-Horizon.git
cd Red-Horizon

# Create build directory
mkdir build && cd build

# Configure and build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```
